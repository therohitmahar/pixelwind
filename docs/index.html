Installation

Use Pixelwind in your project in two ways:

1. CDN (HTML)

<link rel="stylesheet" href="https://unpkg.com/pixelwind@latest/index.css">

2. npm (React, Vite, etc.)

npm install pixelwind
import "pixelwind/index.css";
